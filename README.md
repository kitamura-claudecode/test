# アラームアプリ

シンプルなWebベースのアラームアプリです。

## 機能

- リアルタイム時計表示
- アラームの追加（時刻・ラベル）
- アラームのオン/オフ切り替え
- アラームの削除
- アラーム発火時にビープ音と通知画面を表示
- アラーム設定はブラウザの `localStorage` に保存（リロードしても維持）

## 使い方

1. `index.html` をブラウザで開く
2. 時刻を選択し、必要であればラベルを入力して **追加** をクリック
3. 設定した時刻になるとアラームが鳴ります
4. **止める** ボタンで停止

## 技術スタック

- HTML / CSS / JavaScript（バニラ）
- Web Audio API（アラーム音）
- localStorage（データ永続化）
